## Stinger Royalty-Free EULA 1.0

Этот EULA предназначен для проектов ROMAN ILYIN, которые должны быть бесплатными для личных, внутренних, открытых и коммерческих Конечных продуктов, без роялти, но при этом запрещать standalone-продажу, перепродажу, платное перераспространение или отдельную коммерциализацию Актива или Производных активов.

Лицензия является source-available и royalty-free для Конечных продуктов. Она не является open source лицензией, одобренной OSI, поскольку ограничивает standalone-перепродажу и standalone-коммерциализацию.

## Как использовать в проектном репозитории

Скопируйте текст EULA в корень проекта как:

```text
LICENSE.md
```

Затем добавьте project-specific файл:

```text
NOTICE.md
```

Файл `NOTICE.md` должен указывать название проекта, канонический репозиторий, правообладателя и обязательное уведомление об атрибуции.
