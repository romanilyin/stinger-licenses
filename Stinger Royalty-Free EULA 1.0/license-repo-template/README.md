## Stinger Royalty-Free EULA 1.0

This EULA is intended for ROMAN ILYIN projects that should be free for personal, internal, open, and commercial End Products, royalty-free, while prohibiting standalone sale, resale, paid redistribution, or standalone commercialization of the Asset or Derivative Assets.

It is source-available and royalty-free for End Products. It is not an OSI-approved open source license because it restricts standalone resale and standalone commercialization.

## How to use in a project repository

Copy the EULA text into the project root as:

```text
LICENSE.md
```

Then add a project-specific:

```text
NOTICE.md
```

The `NOTICE.md` file should identify the project name, canonical repository, copyright holder, and required attribution notice.
