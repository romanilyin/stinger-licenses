# Как применить этот многоязычный overlay

Архив разложен как корень репозитория `stinger-licenses`.

Рекомендуемый порядок:

```bash
# из локального клона https://github.com/romanilyin/stinger-licenses
rsync -av --exclude .git /path/to/unpacked/stinger-licenses/ ./

git status
git add .
git commit -m "Add convenience translations for Stinger Royalty-Free EULA 1.0"
git push
```

Не используй `--delete`, если специально не хочешь удалить файлы, которых нет в этом архиве.

## Приоритет языков

Английские `.md` файлы остаются файлами по умолчанию для удобного чтения на GitHub.
Русские `.ru.md` файлы являются основными и имеют приоритет.
Все остальные языковые файлы являются только переводами для удобства.
Если перевод противоречит русской версии, отличается от неё или толкуется иначе, применяется русская версия.

## Важное различие

Корневые `LICENSE.<lang>.md` описывают разрешение копировать и сохранять тексты лицензий в этом репозитории.
Полные переводы самой EULA лежат в `LICENSES/`.

Для проектных репозиториев `CanonicalPath`, `sgg-perfmeter` и `sgg-unity-mcp-gateway` готовые overlay-файлы лежат в `examples/repo-overlays/`.
