# Stinger Licenses

Этот репозиторий хранит тексты лицензий, автором которых является **ROMAN ILYIN**.

Файлы с базовым именем `.md` оставлены на английском, чтобы GitHub и большинство читателей открывали их сразу в удобном виде. Русские файлы `.ru.md` являются основными и имеют приоритет. При расхождении, противоречии или разном толковании английской и русской версий применяется русская версия.

## Основная лицензия

| Лицензия | SPDX identifier | Английский справочный текст | Основной русский текст |
|---|---|---|---|
| Stinger Royalty-Free EULA 1.0 | `LicenseRef-Stinger-Royalty-Free-EULA-1.0` | `LICENSES/LicenseRef-Stinger-Royalty-Free-EULA-1.0.md` | `LICENSES/LicenseRef-Stinger-Royalty-Free-EULA-1.0.ru.md` |

## Stinger Royalty-Free EULA 1.0

`Stinger Royalty-Free EULA 1.0` предназначена для проектов ROMAN ILYIN, которые должны быть бесплатными для личных, внутренних, открытых и коммерческих Конечных продуктов, без роялти, но при этом должны запрещать standalone-продажу, перепродажу, платное перераспространение или отдельную коммерциализацию Актива или Производных активов.

Лицензия является source-available и royalty-free для Конечных продуктов. Она не является open source лицензией, одобренной OSI, потому что ограничивает standalone-перепродажу и standalone-коммерциализацию.

## Как использовать в проектном репозитории

Для проекта ROMAN ILYIN, распространяемого по этой лицензии, скопируйте оба текста лицензии в корень проекта:

```text
LICENSE.md       # английский справочный текст для удобства чтения
LICENSE.ru.md    # основной русский текст
NOTICE.md        # project-specific notice, обычно английский по умолчанию
NOTICE.ru.md     # project-specific русский notice, рекомендуется
```

Готовые примеры для отдельных репозиториев лежат в `examples/repo-overlays/`.

## SPDX

Для файлов кода и package metadata используйте:

```text
SPDX-License-Identifier: LicenseRef-Stinger-Royalty-Free-EULA-1.0
```

Для Unity `package.json`:

```json
{
  "license": "LicenseRef-Stinger-Royalty-Free-EULA-1.0"
}
```

## Разрешение на использование текстов лицензий

Корневой `LICENSE.md` — английская справочная версия разрешения на копирование и сохранение текстов лицензий из этого репозитория. Корневой `LICENSE.ru.md` — основная русская версия такого разрешения.

Это разрешение применяется только к документам с текстами лицензий, примерам notice-файлов и README-вставкам в этом репозитории. Оно не предоставляет прав на исходный код, ассеты, packages, tools, repositories, trademarks, logos или иные материалы, лицензируемые по этим текстам лицензий.
